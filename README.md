Aplicação simples em Java Swing usando o padrão Mediator.

O programa abre duas janelas, cada uma com duas caixas de texto e um botão. Ao clicar no botão de uma janela, os textos digitados nela são copiados para a outra janela.

## Padrão utilizado

Foi utilizado o padrão Mediator para centralizar a comunicação entre as janelas. Assim, uma janela não precisa conhecer diretamente a outra.
